1. O que é Markdown?

De acordo com o guia do markdown:

"the simple and easy-to-use markup language you can use to format virtually any document"

1.0.1. markup language

Você adiciona marcas em um texto para criar itens como títulos, listas, links…

1.0.2. the simple and easy-to-use

Usando símbolos simples como #, -, *, criamos estrutura nos textos.

1.0.3. GitHub

Além disso é usado no GitHub que possui uma documentação.

1.1. Editor

https://dillinger.io

1.2. Estrutura básica de uma lista de compras

Primeiro vamos inserir um título: 

# Compras 10/02/2025

Os itens vão ficar em uma lista não ordenada, marcadores (-, *, +): 

# Compras 10/02/2025

- Maçã
- Pão
- Leite
- Ovos

1.3. Exercício

  1. Crie a sua lista de compras em https://dillinger.io, adicione título e 8 itens de supermercado como lista não-ordenada.

1.4. Lista ordenada

Use números seguidos de um ponto para criar uma lista ordenada. 

# Compras 10/02/2025

1. Maçã
2. Pão
3. Leite
4. Ovos

1.5. Exercício

  1. Altere a lista do exercício anterior, coloque os itens em uma ordem de importância, começando do mais importante até o menos importante.

1.6. Subtítulos

Podemos criar subtítulos usando mais #: 

# Lista de compras 10/02/2025

## Hortifrutigranjeiro

1. Maçã
1. Ovos

## Padaria

1. Pão

## Laticínios

1. Leite

1.7. Exercício

  1. Altere o documento do exercício anterior, separe os itens em setores do supermercado, use ao menos 4 setores com 2 itens em cada setor, veja os setores em https://www.infoprice.co/blog/secao-de-supermercado/.

1.8. Caixas de seleção

Vários editores Markdown aceitam checkboxes para caixas de seleção, use - [ ] ou somente [ ] para um item que ainda não foi marcado e - [x] para um item pronto ou marcado: 

# Lista de compras 10/02/2025

## Hortifrutigranjeiro

- [ ] Maçã
- [ ] Ovos

## Padaria

- [x] Pão

## Laticínios

- [ ] Leite

1.9. Tabelas em Markdown

Para criar uma tabela em Markdown usamos:

  • | para separar as colunas
  • - para separar linhas
| Produto | Faber Castell | Leo&Leo |
|--|--|--|
| Lápis 12 cores | 9,06 | 5,67 |

Para alinhar as colunas use:

  • :--- para alinhar à esquerda
  • ---: para alinhar à direita
  • :---: para centralizar
| Produto | Faber Castell | Leo&Leo |
|--|---:|---:|
| Lápis 12 cores | 9,06 | 5,67 |

1.10. Exercício

  1. Pesquise o preço de 3 produtos de sua escolha em pelo menos dois sites diferentes e organize os dados em tabelas Markdown. O nome do produto deve ficar centralizado e os preços devem ficar alinhados à direita.

1.11. Parágrafos

Os parágrafos são textos de uma ou mais linhas e não precisam de marcas. Os parágrafos devem ser separados com uma linha em branco. 

Diário é um caderno, geralmente de caráter íntimo, onde se fazem anotações que
contêm uma narrativa manuscrita, geralmente diária de experiências pessoais
que é organizada pela data de entrada das informações. Um diário pessoal pode
incluir experiências de uma pessoa e/ou pensamentos, sentimentos e segredos.
Alguém que escreve um diário é conhecido como um diarista e ele descreve toda
sua vida ali. Existem diários desenvolvidos para fins institucionais que
desempenham um papel em muitos aspectos da civilização, como por exemplo,
registros governamentais, livros empresariais e registos alegóricos.

1.12. Destaque

Podemos destacar trechos de texto usando ** para negrito e * para itálico. 

*Memorial de Aires* é o último romance de **Machado de Assis**, escrito em **1907** e
 publicado em livro em julho do mesmo ano de sua morte, 1908.

1.13. Exercício

  1. Encontre um assunto na wikipedia e copie dois parágrafos. Marque duas palavras em negrito e duas em itálico. Veja quando usar itálico aqui: https://www12.senado.leg.br/manualdecomunicacao/estilos/italico
